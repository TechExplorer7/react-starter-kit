- TypeScript(型安全な JavaScript)

### フレームワーク / ライブラリ

- React(UI 開発のライブラリ)
- TanStack Query(データフェッチライブラリ)
- Jotai(状態管理ライブラリ)
- Mantine(UI ライブラリ)

### パッケージ管理

- npm(依存関係の管理ツール)

### 開発ツール

- Vite(ビルドツール)
- MSW(API モック)

- ESLint(コードスタイルと品質チェック)

- Jest(ユニットテスト)
- React Testing Library(React コンポーネントテスト)
